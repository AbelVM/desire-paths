## What It Does
Desire Paths is an open-source web application that models how pedestrians naturally walk across real terrain. Users drop origin and destination points on a MapLibre GL map; the app builds a friction surface from visible map tiles, then runs hundreds of agents through an H3 hex grid using Dijkstra gradients and softmax sampling. Agents leave wear trails that later agents prefer, producing emergent desire paths via positive feedback.

Key technical facts:
- H3 resolution 15 (~0.88 m spacing)
- Web Workers for parallel gradient computation
- 341 passing tests (Vitest)
- Built with Vite, MapLibre GL, Deck.gl, h3-js
